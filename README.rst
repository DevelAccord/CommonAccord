LeanJS
======

Yet another starter kit for Universal NodeJS applications using React, Redux, Webpack and Bootstrap 4.

Install
:::::::

    npm install
    
Run (development)
:::::::::::::::::

    make
    
Build (for production)
::::::::::::::::::::::

    make build
    
Build (for production, using docker)
::::::::::::::::::::::::::::::::::::

    make docker-build
    
Production run (using docker)
:::::::::::::::::::::::::::::

    make docker-run
    
Final word
::::::::::

Good luck.
